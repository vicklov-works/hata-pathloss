ABOUT THE WEBSITE

This website is a product of the final year project of some students from the University of Olabisi Onabanjo University, Ibogun chapter, Ogun state, Nigeria. This website is an interface for a machine learning model that predicts path loss.

This website was born out of the idea that creating a web-app that could predict path loss using the okumura-hata model would save non-programmers the stress of having to go through codes before arriving at the results they need.

OKUMURA-HATA MODEL

Okumura-Hata model is a model used for predicting path loss in transmission in external environment. The model, which was formed by Hata, is an empirical formulation based on the data from the Okumura model, and is thus also commonly referred to as the Okumuraâ€“Hata model. The model is used for open, suburban and urban environments.

Open environment is one that a lot of open field and little houses or tall trees that will serve as obstruction during transmission.

Suburban environment is one that has tall houses or trees that are not too closely packed.

Urban environment is one that has tall houses or trees that are so closely packed and this has even greater impact on the transmission power, that is, it increases the path loss.

Parameters used in the model includes frequency (in MegaHertz) which ranges from 150 to 1500, transmitter or base antenna height (in meters) which ranges from 30 to 200, receiver or mobile antenna height (in meters) which ranges from 1 to 10, and, distance between the base and mobile antennas (in kilometers) which has a maximum value of 20km.
